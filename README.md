# Code challenge for Unmind - Client

## Tech Stack:

- ES6
- React
- GraphQL
- npm

## Getting Started

### Installing

To prepare this app for running you have to:

1. Install **node.js**
2. Install [**Unmind Server**](https://github.com/legan-/unmind-server)
3. `git clone git@github.com:legan-/unmind-client.git`
4. `cd unmind-client`
5. `yarn install`

### Running app

`yarn start`
